---
title: List Monitors
description: Retrieve all monitors for your account
---

## Request

```
GET /api/v1/monitors
```

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |

## Response

```json
{
  "monitors": [
    {
      "id": "7",
      "username": "elonmusk",
      "xUserId": "44196397",
      "eventTypes": ["tweet.new", "tweet.reply"],
      "isActive": true,
      "createdAt": "2026-02-24T10:30:00.000Z"
    }
  ],
  "total": 1
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Monitor ID |
| `username` | string | Monitored X username |
| `xUserId` | string | X user ID |
| `eventTypes` | string[] | Subscribed event types |
| `isActive` | boolean | Whether the monitor is active |
| `createdAt` | string | ISO 8601 creation timestamp |
| `total` | number | Total monitor count |

Returns up to 200 monitors.
