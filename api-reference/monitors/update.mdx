---
title: Update Monitor
description: Update event types or toggle a monitor's active state
---

## Request

```
PATCH /api/v1/monitors/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Monitor ID |

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |
| `Content-Type` | Yes | `application/json` |

### Body

```json
{
  "eventTypes": ["tweet.new", "tweet.quote", "tweet.reply"],
  "isActive": true
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `eventTypes` | string[] | No | Updated event types (min 1) |
| `isActive` | boolean | No | Enable or disable the monitor |

At least one field is required.

## Response

<Tabs>
  <Tab title="200 OK">
    ```json
    {
      "id": "7",
      "username": "elonmusk",
      "xUserId": "44196397",
      "eventTypes": ["tweet.new", "tweet.quote", "tweet.reply"],
      "isActive": true,
      "createdAt": "2026-02-24T10:30:00.000Z"
    }
    ```
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found" }
    ```
  </Tab>
</Tabs>

<Info>
  Toggling `isActive` automatically registers or unregisters the account with the real-time stream.
</Info>
