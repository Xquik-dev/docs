---
title: Delete Monitor
description: Remove a monitor and stop tracking the X account
---

## Request

```
DELETE /api/v1/monitors/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Monitor ID |

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |

## Response

<Tabs>
  <Tab title="200 OK">
    ```json
    { "success": true }
    ```
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found" }
    ```
  </Tab>
</Tabs>

The monitor is soft-deleted and the X account is automatically unregistered from the real-time stream.
