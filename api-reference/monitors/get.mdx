---
title: Get Monitor
description: Retrieve a single monitor by ID
---

## Request

```
GET /api/v1/monitors/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Monitor ID |

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |
