# API Overview

The Xquik REST API v1 provides programmatic access to monitors, events, and webhooks.

## Base URL

```
https://xquik.com/api/v1
```

## Authentication

All API requests require an API key passed via the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

Generate API keys from your [dashboard](https://xquik.com).

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/monitors` | List all monitors |
| POST | `/monitors` | Create a monitor |
| GET | `/monitors/:id` | Get monitor details |
| PATCH | `/monitors/:id` | Update a monitor |
| DELETE | `/monitors/:id` | Delete a monitor |
| GET | `/monitors/:id/events` | List monitor events |
| GET | `/webhooks` | List all webhooks |
| POST | `/webhooks` | Create a webhook |
| GET | `/webhooks/:id` | Get webhook details |
| PATCH | `/webhooks/:id` | Update a webhook |
| DELETE | `/webhooks/:id` | Delete a webhook |

## Rate Limits

API requests are rate-limited to 10 requests per second with a burst of 20.
