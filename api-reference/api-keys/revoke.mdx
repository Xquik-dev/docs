---
title: Revoke API Key
description: Permanently deactivate an API key
---

<Note>
  This endpoint requires **session cookie** authentication only (dashboard).
</Note>

## Request

```
DELETE /api/v1/api-keys/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | API key ID |

## Response

<Tabs>
  <Tab title="200 OK">
    ```json
    { "success": true }
    ```
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found" }
    ```
  </Tab>
</Tabs>

<Warning>
  Revocation is immediate and permanent. All requests using the revoked key will return `401`.
</Warning>
