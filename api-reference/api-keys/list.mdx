---
title: List API Keys
description: Retrieve all API keys for your account
---

<Note>
  This endpoint requires **session cookie** authentication only (dashboard).
</Note>

## Request

```
GET /api/v1/api-keys
```

## Response

```json
{
  "keys": [
    {
      "id": "42",
      "name": "Production",
      "prefix": "a1b2",
      "isActive": true,
      "createdAt": "2026-02-24T10:30:00.000Z",
      "lastUsedAt": "2026-02-24T14:00:00.000Z"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Key ID |
| `name` | string | Display name |
| `prefix` | string | First 4 characters after `xqk_` |
| `isActive` | boolean | Whether the key is active |
| `createdAt` | string | ISO 8601 creation timestamp |
| `lastUsedAt` | string | ISO 8601 last usage timestamp (omitted if never used) |

Returns up to 200 keys.
