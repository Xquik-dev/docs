---
title: Update Webhook
description: Update URL, event types, or toggle a webhook's active state
---

## Request

```
PATCH /api/v1/webhooks/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Webhook ID |

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |
| `Content-Type` | Yes | `application/json` |

### Body

```json
{
  "url": "https://new-server.com/webhook",
  "eventTypes": ["tweet.new"],
  "isActive": false
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `url` | string | No | New HTTPS endpoint URL |
| `eventTypes` | string[] | No | Updated event types (min 1) |
| `isActive` | boolean | No | Enable or disable the webhook |

At least one field is required.
