---
title: List Webhooks
description: Retrieve all webhook endpoints for your account
---

## Request

```
GET /api/v1/webhooks
```

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |

## Response

```json
{
  "webhooks": [
    {
      "id": "15",
      "url": "https://your-server.com/webhook",
      "eventTypes": ["tweet.new", "tweet.reply"],
      "isActive": true,
      "createdAt": "2026-02-24T10:30:00.000Z"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Webhook ID |
| `url` | string | Delivery endpoint |
| `eventTypes` | string[] | Subscribed event types |
| `isActive` | boolean | Whether the webhook is active |
| `createdAt` | string | ISO 8601 creation timestamp |

Returns up to 200 webhooks. The `secret` is never included in list responses.
