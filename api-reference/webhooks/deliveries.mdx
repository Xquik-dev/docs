---
title: List Deliveries
description: View delivery attempts for a specific webhook
---

<Note>
  This endpoint requires **API key** authentication only.
</Note>

## Request

```
GET /api/v1/webhooks/{id}/deliveries
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Webhook ID |

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |

## Response

```json
{
  "deliveries": [
    {
      "id": "501",
      "status": "success",
      "streamEventId": "9001",
      "attempts": 1,
      "lastStatusCode": 200,
      "createdAt": "2026-02-24T14:22:01.000Z",
      "deliveredAt": "2026-02-24T14:22:02.000Z"
    },
    {
      "id": "502",
      "status": "failed",
      "streamEventId": "9002",
      "attempts": 3,
      "lastStatusCode": 500,
      "lastError": "Internal Server Error",
      "createdAt": "2026-02-24T14:25:00.000Z"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Delivery ID |
| `status` | string | `"pending"`, `"success"`, or `"failed"` |
| `streamEventId` | string | Associated stream event ID |
| `attempts` | number | Total delivery attempts |
| `lastStatusCode` | number | HTTP status from your endpoint (optional) |
| `lastError` | string | Error message from last attempt (optional) |
| `createdAt` | string | ISO 8601 creation timestamp |
| `deliveredAt` | string | ISO 8601 successful delivery timestamp (optional) |

Returns up to 100 deliveries, most recent first.
