---
title: Get Account
description: Retrieve account info, subscription status, and usage
---

<Note>
  This endpoint requires **API key** authentication only.
</Note>

## Request

```
GET /api/v1/account
```

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your API key |

## Response

<Tabs>
  <Tab title="200 OK">
    ```json
    {
      "plan": "active",
      "monitorsAllowed": 1,
      "monitorsUsed": 0,
      "currentPeriod": {
        "start": "2026-02-01T00:00:00.000Z",
        "end": "2026-03-01T00:00:00.000Z",
        "capMicrocents": "10000000000",
        "usedMicrocents": "1500000"
      }
    }
    ```

    | Field | Type | Description |
    |-------|------|-------------|
    | `plan` | string | `"active"` or `"inactive"` |
    | `monitorsAllowed` | number | Max monitors for your plan |
    | `monitorsUsed` | number | Currently active monitors |
    | `currentPeriod` | object | Current billing period (omitted if no active subscription) |
    | `currentPeriod.capMicrocents` | string | Monthly usage cap in microcents ($10 = `"10000000000"`) |
    | `currentPeriod.usedMicrocents` | string | Usage consumed this period |
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated" }
    ```
  </Tab>
</Tabs>

### Usage Cap

Every subscriber has a $10/month usage cap. Costs are tracked in **microcents** (1 microcent = $0.000001):

| Operation | Cost (microcents) |
|-----------|-------------------|
| Tweet search | 150 |
| User lookup | 180 |
| Follower fetch | 150 |
| Verified follower fetch | 300 |
| Follow check | 150 |
