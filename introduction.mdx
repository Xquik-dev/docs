# Welcome to Xquik

Xquik is an X (Twitter) real-time data platform providing extraction tools, giveaway draws, account monitoring, and a developer API.

## Features

- **8 Extraction Tools** — followers, following, retweeters, likers, quoters, replies, list members, verified followers
- **Giveaway Draws** — transparent, auditable random draws with archived tweet snapshots
- **Account Monitoring** — real-time tracking of follower changes, tweet activity, and engagement
- **REST API v1** — programmatic access to all platform features
- **HMAC Webhooks** — real-time event delivery with cryptographic verification
- **MCP Server** — AI agent integration with 9 tools
